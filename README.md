# Fake News Detection using Machine Learning

This project detects whether news articles are REAL or FAKE using NLP and Machine Learning.

## Technologies Used
Python
Scikit-learn
TF-IDF Vectorizer
PassiveAggressiveClassifier

## Dataset
The dataset contains 7796 news articles labeled REAL or FAKE.

## Accuracy
The model achieves around 92% accuracy.

## How to Run

Install libraries:

pip install -r requirements.txt

Run:

python fake_news_detection.py
